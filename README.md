- Installation
- Keyboard/Trackpad
- Battery Information
- Intel Bluetooth
- Thunderbolt (no Hot-Plug)
- HiDPI Screen

**What doesn't work (yet)**
- Intel AX200 WiFi
- Brightness control
- Touch screen
- Intel Iris Plus G7 (recognized no framebuffer loaded)
